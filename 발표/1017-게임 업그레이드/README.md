## 🎯 1. 기본 게임 구조 요약

* **게임 룰:**
  플레이어가 번갈아 1~10 사이의 수를 입력 → 누적합이 **100 이상이 되면 입력한 사람이 승리**
* **기존 구성:**
  콘솔 텍스트 기반, 숫자 입력만으로 진행, `gotoxy()` 등으로 간단히 출력

---

## 🚀 2. 업그레이드 방향 (전체 플랜)

| 구분              | 세부 목표               | 구현 아이디어                                                                                |
| --------------- | ------------------- | -------------------------------------------------------------------------------------- |
| **비주얼(인트로·폰트)** | 시작 시 타이틀 화면 + 폰트 효과 | `Sleep()`·`Beep()`로 타이틀 점멸, ASCII 아트(“100 Game!”) 출력, 색상(`SetConsoleTextAttribute`) 변경 |
| **사운드**         | 입력 및 승패 시 효과음       | `Beep(440,200)` / `Beep(880,150)` 등 다양한 주파수 조합으로 멜로디, 승리 시 짧은 팬파레                      |
| **UX 개선**       | 턴 표시·누적합 시각화        | “Player 1 ▶” “Player 2 ▶” 번갈아 표시, 누적바(`■■■■■■`) 길이로 진행상태 시각화                           |
| **AI 추가**       | 혼자 할 때 컴퓨터와 대전      | `rand()%10+1` 랜덤 선택 or 간단한 전략(`11 - player_input`) 적용                                  |
| **UI 개선**       | 메뉴 선택 및 재시작         | `메뉴: 1.시작  2.설명  3.종료` 구조, 게임 끝나면 “다시 하기?” Y/N                                         |
| **테마 확장**       | 배경 스토리 추가           | “우주에서 100을 먼저 모으는 자가 승리한다!” 식으로 분위기 설정                                                 |
| **데이터 로깅**      | 최고 점수 기록            | `record.txt`에 승패 횟수 저장, `ifstream/ofstream` 사용                                         |
| **안정성**         | 예외 입력 처리            | 1~10 아닌 숫자/문자 입력 시 오류 메시지 후 재입력 유도                                                     |

## 🧩 3. 코드 구조 리팩토링 제안

| 구조             | 내용                                                                          |
| -------------- | --------------------------------------------------------------------------- |
| **main() 최소화** | 메뉴 → 게임루프 → 결과 출력 흐름                                                        |
| **함수 분리**      | `showIntro()`, `drawBoard(sum)`, `playerTurn()`, `aiTurn()`, `showWinner()` |
| **전역 상수**      | `#define MAX_SUM 100`, `#define MIN_NUM 1`, `#define MAX_NUM 10`            |
| **전역 변수 최소화**  | playerScore[], turn, sum 등은 구조체로 묶기                                         |
| **색상 헬퍼 함수**   | `void setColor(int fg, int bg)` → 콘솔 출력 강조용                                 |

## 🔊 4. 연출 요소 구체 예시

### 인트로

```cpp
void showIntro() {
    system("cls");
    setColor(14,0);  // 노란 글씨
    cout << "\n\n\t=== 100 넘기기 게임 ===\n";
    setColor(7,0);
    Sleep(500);
    cout << "\tPress any key to start...\n";
    Beep(440, 200);
    getch();
}
```

### 효과음

```cpp
void playWinSound() {
    Beep(523,150); Beep(659,150); Beep(784,200);
}
```

### 누적 시각화

```cpp
void drawBar(int sum) {
    cout << "\n누적합: " << sum << " / 100\n";
    cout << "[";
    int blocks = sum / 5;
    for(int i=0; i<blocks; i++) cout << "■";
    for(int i=blocks; i<20; i++) cout << " ";
    cout << "]\n";
}
```
